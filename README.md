# Wow Web Music Player 🎧✨

A visually rich music player built with **Vite + React + TypeScript** and deployed to **GitHub Pages**.

- Glassmorphism UI
- Animated gradient/aurora background
- Web Audio API visualizer with graceful fallback
- Keyboard shortcuts + mobile-friendly layout
- Local royalty-safe demo tracks generated by script

## Features

- ▶️ Play / Pause
- ⏮ / ⏭ Previous / Next track
- ⏱ Seek bar + live progress + total duration
- 🔊 Volume control
- 🎼 Playlist with 3 local demo tracks:
  - Neon Drift
  - Violet Pulse
  - Sunrise Glide
- ⌨️ Keyboard shortcuts:
  - `Space` — play/pause
  - `←/→` — seek ±5s
  - `N` / `P` — next/previous track
  - `M` — mute/unmute
- 📱 Responsive mobile UI
- 📊 Audio visualizer (Web Audio API), fallback message if unsupported

## Tech Stack

- React 19
- TypeScript
- Vite 8
- GitHub Actions (Pages deploy)

---

## Local Run

```bash
git clone https://github.com/SDV-G-Deploy/wow-web-music-player.git
cd wow-web-music-player
npm install
npm run dev
```

Open: http://localhost:5173

---

## Build

```bash
npm run build
npm run preview
```

---

## Demo Audio Generation

All demo tracks are generated locally (sine-based synthesis, no copyright risk):

```bash
npm run generate:audio
```

Script: `scripts/generate-demo-audio.mjs`

---

## Deployment

Deployment is automated with `.github/workflows/deploy.yml`:

1. Push to `main`
2. GitHub Actions builds Vite app
3. Artifact is deployed to GitHub Pages
